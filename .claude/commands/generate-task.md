# タスク生成コマンド

## 指示
`$ARGUMENTS`のファイル内容に従って、新しいタスクのための`.claude/tasks/task-[タスク内容].md`を生成してください。
タスクを複数に分解する必要がある場合は、複数のタスクファイルを生成してください。

## task.md の形式
```markdown
# Task: [タスク名]

## 実装計画
### Phase 1: 調査・設計
- [ ] 既存コード調査
- [ ] 設計方針決定

### Phase 2: 実装
- [ ] テストコード作成
- [ ] 機能実装

### Phase 3: 検証
- [ ] テスト実行
- [ ] レビュー対応
- [ ] ビルド実行
- [ ] ビルドエラー解消

## 進捗メモ
作業進捗を随時更新
